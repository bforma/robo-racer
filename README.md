# Robo Racer

[![Build Status](https://travis-ci.org/bforma/robo-racer.svg?branch=master)](https://travis-ci.org/bforma/robo-racer)

## Getting started

```
git clone git@github.com:bforma/robo-racer.git
bundle install
rails server
```

And point your browser to http://localhost:3000

## Running tests

`rspec spec`
