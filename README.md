## Getting started

### Prerequisites

Redis (`brew install redis`)
Mongodb (`brew install mongodb`)

### Installation

```
git clone git@github.com:bforma/robo-racer.git
bundle install
rails server
```

And point your browser to http://localhost:3000

## Running tests

`rspec spec`
